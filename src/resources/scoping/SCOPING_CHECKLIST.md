# Scoping Checklist

- Objectives defined
- Scope in/out clarified
- Stakeholders identified
- Timeframe agreed
- Constraints and budgets captured
- Allowed/blocked tools listed
- Unknowns and assumptions documented
- Success criteria and verification steps defined
- Risks identified with mitigations
- Initial decomposition into sub-questions
- Tool selection rationale
- Estimated cost/time vs budgets
- Plan critique completed (go/no-go)
